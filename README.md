# nobodywho.ooo

Built with 11ty, Alpine.js and Tailwind. 

## Getting started

1. Clone the repo
2. Run `npm install`

## Develop

Run `npm start`

## Build

Run `npm run build`

## Nix Flake

There's also a nix flake.

To build the website: `nix build`

To build and serve the website built w/ nix: `nix run`

To run a hot-reloading dev server: `nix run '.#dev'`
